Detailed Description

The QAbstractFontEngine class is the base class for font engine plugins in Qt for Embedded Linux.

This function is under development and subject to change.

Since
4.3

QAbstractFontEngine is implemented by font engine plugins through QFontEnginePlugin.

See also
QFontEnginePlugin, QFontEngineInfo

enum QAbstractFontEngine::Capability

This enum describes the capabilities of a font engine.

  • CanRenderGlyphs_Gray The font engine can render individual glyphs into 8 bpp images.
  • CanRenderGlyphs_Mono The font engine can render individual glyphs into 1 bpp images.
  • CanRenderGlyphs The font engine can render individual glyphs into images.
  • CanOutlineGlyphs The font engine can convert glyphs to painter paths.

◆ FontProperty

enum QAbstractFontEngine::FontProperty

This enum describes the properties of a font provided by a font engine.

  • Ascent The ascent of the font, specified as a 26.6 fixed point value.
  • Descent The descent of the font, specified as a 26.6 fixed point value.
  • Leading The leading of the font, specified as a 26.6 fixed point value.
  • XHeight The 'x' height of the font, specified as a 26.6 fixed point value.
  • AverageCharWidth The average character width of the font, specified as a 26.6 fixed point value.
  • LineThickness The thickness of the underline and strikeout lines for the font, specified as a 26.6 fixed point value.
  • UnderlinePosition The distance from the base line to the underline position for the font, specified as a 26.6 fixed point value.
  • MaxCharWidth The width of the widest character in the font, specified as a 26.6 fixed point value.
  • MinLeftBearing The minimum left bearing of the font, specified as a 26.6 fixed point value.
  • MinRightBearing The maximum right bearing of the font, specified as a 26.6 fixed point value.
  • GlyphCount The number of glyphs in the font, specified as an integer value.
  • CacheGlyphsHint A boolean value specifying whether rendered glyphs should be cached by Qt.
  • OutlineGlyphsHint A boolean value specifying whether the font engine prefers outline drawing over image rendering for uncached glyphs.

) const
pure virtual

Implemented in subclasses to convert the characters specified by string and length to glyph indicies, using flags.

The glyph indicies should be returned in the glyphs array provided by the caller. The maximum size of glyphs is specified by the value pointed to by numGlyphs. If successful, the subclass implementation sets the value pointed to by numGlyphs to the actual number of glyph indices generated, and returns true. Otherwise, e.g. if there is not enough space in the provided glyphs array, it should set numGlyphs to the number of glyphs needed for the conversion and return false.
