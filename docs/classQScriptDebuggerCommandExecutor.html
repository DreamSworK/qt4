Detailed Description

The QScriptDebuggerCommandExecutor applies debugger commands to a back-end.

Since
4.5
Warning
This function is not part of the public interface.

The execute() function takes a command (typically produced by a QScriptDebuggerFrontend) and applies it to a QScriptDebuggerBackend.
