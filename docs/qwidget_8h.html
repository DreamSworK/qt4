◆ QWIDGETSIZE_MAX

#define QWIDGETSIZE_MAX   ((1<<24)-1)

Defines the maximum size for a QWidget object.

The largest allowed size for a widget is QSize(QWIDGETSIZE_MAX, QWIDGETSIZE_MAX), i.e. QSize (16777215,16777215).
